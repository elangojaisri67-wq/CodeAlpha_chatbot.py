# Basic Chatbot

## Description
This is a simple rule-based chatbot created using Python. 
It responds to user inputs based on predefined conditions.

## Features
- Responds to greetings like "hello"
- Answers basic questions
- Interactive loop until user exits

## Technologies Used
Python

## How to Run
1. Download the file
2. Run using:
   python chatbot.py
